\doxysubsection{Detailed Description}
Class that lets the user input commands for the drone. 

The \mbox{\hyperlink{classCommand}{Command}} class is a user input module that is used to manually input messages for the system to send to the drone. The commands are identified by an ID which is an integer number that goes from 0 to 5.
