Detailed Description

Class that lets the user input commands for the drone.

The Command class is a user input module that is used to manually input messages for the system to send to the drone. The commands are identified by an ID which is an integer number that goes from 0 to 5.

◆ send_trajectory()

void Command::send_trajectory ( )

The send_trajectory method. This method is called when the user insert a serie of point for the drone to follow (or uses the test points). It creates a Catmull-Rom curve interpolating the control points and sends int_points_num intermediate points to the drone one after the other.
